# UI Theme Rules

This document records the current visual rules for the simplified `jobvisualizer` page.

## Background

- Base page background: `#f6eee3`

## Palette

Primary palette:

- `#f6eee3` rgb `(246, 238, 227)`
- `#eee7d7` rgb `(238, 231, 215)`
- `#e5decf` rgb `(229, 222, 207)`
- `#e5cbba` rgb `(229, 203, 186)`
- `#d9bda5` rgb `(217, 189, 165)`

## Usage

- Page background uses `#f6eee3`
- Softer surfaces use `#eee7d7`
- Panel blocks and chips use `#e5decf`
- Accent surfaces and active states use `#e5cbba`
- Borders and stronger emphasis use `#d9bda5`

## Typography

- Primary font: `Cormorant Garamond`
- This font is used for both display and interface text in the current theme

## Intent

The page should feel:

- calmer
- lighter
- less visually noisy
- more editorial and minimal

The palette is intentionally warm and low-contrast compared with the earlier graph-heavy interface.
